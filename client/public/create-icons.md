# PWA Icons

צריך ליצור את האייקונים הבאים:

1. `pwa-192x192.png` - 192x192 pixels
2. `pwa-512x512.png` - 512x512 pixels  
3. `apple-touch-icon.png` - 180x180 pixels

האייקונים צריכים להיות:
- עיצוב מינימליסטי בסגנון origami
- צבעים: coral (#FFB4A2) ו-peach (#FFE5D9)
- סמל של ספר או כובע גמר
- רקע עם גרדיאנט עדין

ניתן להשתמש בכלי כמו:
- Figma
- Canva
- או כל כלי עיצוב אחר

לאחר יצירת האייקונים, יש להציב אותם בתיקייה `public/`.
