# לוגו ומשאבים

## הוספת לוגו

1. שים את קובץ הלוגו שלך כאן: `logo.png`
2. או שנה את השם – אם השתמשת בקובץ אחר (למשל `my-logo.png`), עדכן את הנתיב בהגדרות העסק

## נתיב ברירת מחדל

המערכת מחפשת: `/assets/logo.png`

קבצים בתיקיית `public/` נטענים ישירות, ולכן:
- `public/assets/logo.png` → זמין בכתובת `/assets/logo.png`

## פורמטים מומלצים

- PNG עם רקע שקוף (מומלץ)
- SVG (גם נתמך)
- גודל מומלץ: 200x80 פיקסלים בערך
